# Smart Backup Script

This Bash script automates daily backups of a specified directory, saves them with a timestamp, logs the process, and deletes old backups (older than 7 days).

## Features

- Creates `.tar.gz` archives of any folder
- Logs backup activity
- Automatically removes old backups
- Easy to set up and run

## Usage

Run the script:

```bash
./smart-backup.sh
